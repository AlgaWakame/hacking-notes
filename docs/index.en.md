## What this site is for

This knowledge base is designed to help with:

- organizing security notes;
- reviewing offensive security concepts;
- documenting authorized labs;
- building reusable testing methodology;
- writing better vulnerability reports;
- preparing for security certifications;
- keeping public notes clean and safe.
